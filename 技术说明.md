# STM32 机械臂下位机技术说明

## 1. 功能定位

本项目面向三线绳驱连续体柔性机械臂。STM32 下位机负责进给/回撤、左右弯曲、上下弯曲等确定性低层控制，并把手柄输入或上位机速度命令转换为三路绳驱电机的速度目标。

STM32 下位机主要负责：

- PS2 手柄输入解析
- Jetson/PC 串口命令解析
- 三线绳驱电机速度混控
- 编码器速度环
- 三路 IMU 姿态采集与上报
- 夹爪开合输出
- 急停、使能、通信超时保护

Jetson 负责目标识别、任务状态机、无人机协同、机械臂外环位置控制。STM32 不直接执行无人机任务规划。

## 2. 控制链路

```text
PS2 / USART3
    -> 虚拟摇杆 feed/x/y
    -> 图案/手动/上位机控制融合
    -> 三电机速度混控
    -> 目标编码器增量
    -> 编码器速度 PI + 前馈
    -> TIM1 PWM + TB6612 方向引脚
```

当前编码器用于速度反馈，尚未实现机械臂累计位置闭环。Jetson 外环可以通过连续发送 `J,feed,x,y` 做位置误差闭环。

## 3. PS2 按键分配

| 按键 | 功能 |
| --- | --- |
| 左摇杆 Y | 进给/后退 |
| 右摇杆 X | 左右弯曲 |
| 右摇杆 Y | 上下弯曲 |
| 方向键 | 替代右摇杆方向控制 |
| 三角 | 绘制三角形 |
| 圆圈 | 绘制圆形 |
| 方块 | 绘制方形 |
| 叉 | 绘制叉形 |
| L1 | 夹爪打开 |
| R1 | 夹爪闭合 |
| L2 | 禁止机械臂电机输出 |
| R2 | 使能机械臂电机输出 |
| SELECT + START | 急停锁存 |
| START + R2 | 清除急停并使能 |
| L3 + R3 | 软件回零/复位速度环 |

急停锁存后，电机输出和夹爪输出都会关闭。

## 4. 引脚分配

| 功能 | 引脚 |
| --- | --- |
| PS2 DAT/CMD/CS/CLK | PB12/PB13/PB14/PB15 |
| 电机 PWM | PA8/PA9/PA10, TIM1_CH1/2/3 |
| 电机方向 | PA2/PA3, PA4/PA5, PB0/PB1 |
| 编码器 | PA0/PA1, PA6/PA7, PB6/PB7 |
| IMU1 | PB8/PB9 |
| IMU2 | PA11/PA12 |
| IMU3 | PB4/PB5 |
| Jetson 串口 | PB10/PB11, USART3 |
| 夹爪打开输出 | PB2 |
| 夹爪闭合输出 | PB3 |

PB3 默认被 JTAG 占用，程序初始化时关闭 JTAG、保留 SWD。夹爪驱动接线前必须确认 PB2/PB3 与实际硬件无冲突。

## 5. USART3 协议

串口参数：`115200 8N1`。

主机到 STM32：

| 命令 | 功能 |
| --- | --- |
| `J,feed,x,y` | 机械臂速度命令，范围 `-100..100` |
| `P,T` | 画三角形 |
| `P,X` | 画叉 |
| `P,O` | 画圆 |
| `P,S` | 画方形 |
| `S` | 停止上位机速度命令 |
| `E` | 急停锁存 |
| `A,0` | 禁止电机输出 |
| `A,1` | 清急停并使能 |
| `G,0` | 夹爪闭合 |
| `G,100` | 夹爪打开 |
| `G,50` | 停止夹爪输出 |
| `G,O/C/S` | 夹爪打开/闭合/停止别名 |
| `H` | 软件回零/复位速度环 |
| `Q` | 查询状态 |

STM32 到主机：

```text
IMU1,t,online,roll,pitch,yaw,temp,press,alt
IMU2,t,online,roll,pitch,yaw,temp,press,alt
IMU3,t,online,roll,pitch,yaw,temp,press,alt
ARM,enabled,estop,gripper,gripper_dir,home_pending
```

`ARM` 字段说明：

- `enabled`: 电机输出使能状态
- `estop`: 急停锁存状态
- `gripper`: 当前夹爪目标开度，`0` 闭合，`100` 打开
- `gripper_dir`: `1` 打开中，`-1` 闭合中，`0` 空闲
- `home_pending`: 软件回零请求标志

## 6. 夹爪实现

当前夹爪实现为双 GPIO 脉冲控制：

- PB2 输出打开脉冲
- PB3 输出闭合脉冲
- 默认脉冲时长约 500 ms

如果实际夹爪是舵机，需要把该部分改成标准 50 Hz 舵机 PWM；如果是电机/继电器/H 桥，当前双 GPIO 方案可以作为基础，但必须根据驱动板确认电平极性。

## 7. 仍需完善

当前下位机仍不是最终无人机任务版本，还建议继续增加：

- 固定定时器中断控制周期，避免 IMU/串口阻塞影响速度环
- 编码器累计位置、行程软限位、真实回零传感器
- 更可靠的带 CRC 帧协议
- TX 环形缓冲或 DMA，减少串口打印阻塞
- IWDG 看门狗
- 电机电流/堵转保护
- 夹爪开合到位反馈
