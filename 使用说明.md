# STM32 机械臂下位机使用说明

## 1. 编译与下载

1. 使用 Keil uVision 打开 `PS2_Demo.uvprojx`。
2. 确认 `USER`、`Hardware`、`System` 目录下的源码已加入工程。
3. 编译并下载到 STM32F103。
4. USART3 连接 Jetson/PC：PB10 为 TX，PB11 为 RX，波特率 `115200`。

## 2. 上电前检查

- 电机驱动电源、电机电源和 STM32 电源接线正确。
- TB6612 方向线和 PWM 线无短路。
- 编码器 A/B 相接线正确。
- 三路 IMU 供电和 I2C 上拉电阻正常。
- 夹爪驱动接到 PB2/PB3 前，先确认驱动输入极性。
- PS2 接收器连接 PB12/PB13/PB14/PB15。
- 首次上电建议断开电机动力电源，只看串口输出和 GPIO 逻辑。

## 3. PS2 手柄操作

### 手动控制

- 左摇杆上下：机械臂进给/后退。
- 右摇杆左右：机械臂左右弯曲。
- 右摇杆上下：机械臂上下弯曲。
- 方向键：替代右摇杆做离散方向控制。

### 图案绘制

保持摇杆回中后：

- 三角：画三角形
- 圆圈：画圆
- 方块：画方形
- 叉：画叉形

移动任意摇杆会取消当前图案。

### 安全与夹爪

- `SELECT + START`：急停锁存。
- `START + R2`：清除急停并重新使能。
- `L2`：禁止电机输出。
- `R2`：使能电机输出。
- `L1`：夹爪打开。
- `R1`：夹爪闭合。
- `L3 + R3`：软件回零/复位速度环。

急停后必须确认机械臂和无人机状态安全，再执行清急停。

## 4. 串口控制

PC 或 Jetson 可直接发送 ASCII 命令，命令以换行结尾。

### 示例

停止：

```text
S
```

使能：

```text
A,1
```

禁止：

```text
A,0
```

急停：

```text
E
```

前进：

```text
J,60,0,0
```

右弯：

```text
J,0,60,0
```

夹爪打开：

```text
G,100
```

夹爪闭合：

```text
G,0
```

查询状态：

```text
Q
```

## 5. Jetson 脚本使用

实时显示 IMU、姿态和键盘控制：

```powershell
python jetson_imu3_realtime_plot.py --port COM5 --baud 115200
```

Linux 下示例：

```bash
python3 jetson_imu3_realtime_plot.py --port /dev/ttyUSB0 --baud 115200
```

上位机键盘映射：

- `W/S`：进给/后退。
- `A/D` 或 `Left/Right`：左右弯曲。
- `Up/Down`：上下弯曲。
- `Space`：停止速度命令。
- `Esc` 或 `E`：急停锁存。
- `Enter` 或 `R`：清急停并使能。
- `Backspace` 或 `F`：禁止电机输出。
- `[`：夹爪闭合。
- `]`：夹爪打开。
- `\`：停止夹爪输出。
- `H`：软件回零/复位速度环。
- `/` 或 `?`：查询 `ARM` 状态。
- `T/1`、`X/2`、`O/3`、`Q/4`：绘制三角形、叉、圆、方形。

自动靠近目标的基础脚本：

```bash
python3 jetson_autonomous_grasp.py --port /dev/ttyUSB0 --baud 115200 --target-distance 0.18
```

## 6. 调试顺序

推荐按下面顺序调试，不要一开始就挂到无人机上：

1. 只接 STM32 和串口，确认启动输出 `IMU*_HEADER` 和 `ARM_HEADER`。
2. 不接电机动力电源，测试 PS2 按键、串口 `A/E/G/Q`。
3. 接夹爪驱动，单独测试 `G,100`、`G,0`。
4. 接电机动力电源，低速发送 `J,20,0,0`，确认方向正确。
5. 测试急停：运动中发送 `E` 或按 `SELECT + START`，电机应立即停止。
6. 调整速度环参数，确认无明显振荡。
7. 接入 Jetson ROS 节点。
8. 最后再进行无人机低空悬停联合测试。

## 7. 故障排查

- 串口没有数据：检查 PB10/PB11 是否交叉连接、波特率是否为 115200。
- PS2 无响应：检查接收器供电和 PB12-PB15 接线，确认手柄处于模拟模式。
- IMU 离线：检查 I2C 上拉、PB4 JTAG 是否释放、传感器地址是否匹配。
- 电机方向反了：优先交换电机线或调整方向引脚逻辑，避免在上位机用负号补偿。
- 急停后无法运动：发送 `A,1` 或按 `START + R2` 清除急停并使能。
- 夹爪动作反了：交换 PB2/PB3 或在代码里调换打开/闭合输出。

## 8. 无人机联调注意事项

- 机械臂动作前，PX4 必须处于稳定悬停。
- Jetson 任务节点必须能随时发送 `E` 急停。
- PX4 Offboard 丢失、姿态角超限、低电压、RC 接管时，Jetson 应立即发送 `E` 或 `A,0`。
- 第一次联合测试必须低空、限幅、限速，并保留人工急停。
